## Objetivo

Praticar manipulação de listas e combinação de dados. Neste exercício, você irá juntar duas listas em uma única lista, mantendo a ordem dos elementos.

## Especificação

### Combinação de Listas

Abra o arquivo `main.py`. Dentro dele, localize a função `resposta`.

A função deverá receber duas listas e retornar uma nova lista contendo todos os elementos da primeira lista seguidos pelos elementos da segunda lista.

Regras:

- Manter a ordem dos elementos
- Primeiro adicionar todos os elementos da primeira lista
- Depois adicionar todos os elementos da segunda lista
- Retornar a nova lista resultante

Exemplos:

- `primeira = [1, 2, 3]`
- `segunda = [4, 5, 6]`
- `resposta = [1, 2, 3, 4, 5, 6]`

**Atenção:** utilize `return`, não `print`.
