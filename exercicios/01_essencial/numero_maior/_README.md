## Objetivo

Praticar o uso de estruturas condicionais e operadores de comparação para analisar valores. Neste exercício, você irá identificar o maior número entre três valores informados.

## Especificação

### Maior de Três Números

Abra o arquivo `main.py`. Dentro dele, localize a função `resposta`.

A função deverá receber três valores numéricos e retornar o maior entre eles.

Regras:

- Comparar os três valores informados
- Identificar qual deles é o maior
- Retornar o maior valor encontrado

Exemplos:

- `resposta(10, 20, 5)` → `20`
- `resposta(7, 3, 9)` → `9`
- `resposta(4, 4, 2)` → `4`

**Atenção:** utilize `return`, não `print`.
