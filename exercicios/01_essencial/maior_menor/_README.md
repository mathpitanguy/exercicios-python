## Objetivo

Praticar o uso de estruturas condicionais e operadores de comparação para analisar múltiplos valores. Neste exercício, você irá identificar o maior e o menor número entre três valores informados.

## Especificação

### Maior e Menor de Três Números

Abra o arquivo `main.py`. Dentro dele, localize a função `resposta`.

A função deverá receber três valores numéricos e retornar uma tupla contendo:

- O maior valor
- O menor valor

Regras:

- Comparar os três valores informados
- Identificar qual é o maior e qual é o menor
- Retornar os valores na ordem: (maior, menor)

Exemplos:

- `resposta(10, 20, 5)` → `(20, 5)`
- `resposta(7, 3, 9)` → `(9, 3)`
- `resposta(4, 4, 2)` → `(4, 2)`

**Atenção:** utilize `return`, não `print`.
